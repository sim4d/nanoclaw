# Antigravity AI Programming Assistant

让 AI 助手具备 Google Antigravity 风格的编程能力。

## 描述

这个 skill 为 AI 助手添加以下能力：

- 🤖 **智能代码生成**：根据需求自动生成代码
- 🔍 **代码分析与优化**：分析现有代码并提供优化建议
- 🐛 **智能调试**：定位和修复代码中的 bug
- 📝 **代码解释**：解释复杂代码的逻辑和功能
- 🔄 **代码重构**：改进代码结构和可读性
- 🧪 **测试生成**：自动生成单元测试
- 💬 **多轮对话编程**：通过对话逐步构建功能

## AI 编程原则

1. **理解需求优先**：在生成代码前，先充分理解需求
2. **代码质量**：生成可读、可维护、高效的代码
3. **错误处理**：添加适当的错误处理和边界条件检查
4. **注释文档**：为关键代码添加清晰的注释
5. **最佳实践**：遵循语言的最佳实践和设计模式
6. **安全性**：考虑代码的安全性，避免常见漏洞

## 对话式编程

使用多轮对话逐步构建复杂功能：

```
用户：帮我创建一个 API 服务器

AI：好的，你希望使用什么框架？
    - Flask (Python)
    - Express (Node.js)
    - Gin (Go)
    - 其他？

用户：用 Express

AI：明白了。这个 API 需要哪些端点？

用户：需要用户注册、登录和获取用户信息

AI：好的，我来创建这些端点...
[生成代码]
```
